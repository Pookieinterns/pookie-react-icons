# Installation

Getting started with **Pookie React Icons** is quick and easy. Follow the instructions below to add it to your project.

## Package Manager

Install the library using your preferred package manager:

### npm
```bash
npm install pookie-react-icons
```

### yarn
```bash
yarn add pookie-react-icons
```

### pnpm
```bash
pnpm add pookie-react-icons
```

## Requirements

- **React** 16.8 or higher.
- **Node.js** 14 or higher recommended.
